# FiniteCurve

## Come Avviare L'app

### Tramite apk:
- Scarica ed installa [l'apk di release](https://1drv.ms/u/s!AgSavy0t5Xu5bhNwd-SmNCt1a4Y?e=ACl2j0) sul proprio dispositivo o un emulatore

### Tramite Node
#### Setup
- Seguire la guida [environment-setup](https://reactnative.dev/docs/environment-setup) selezionando il quickstart di react native

Saranno necessari:
- Node v14
- Java 1.8
- Android Studio
- Emulatore o device fisico
- Opzionale installa yarn ```npm -g install yarn```

### Avvio

- Clona ed antra nella repository con ```bash git clone https://github.com/mmoio/FiniteCurve.git && cd FiniteCurve```

- Installa le dipendenze con ```npm install``` oppure con ```yarn```

- In due terminali all'interno della repository esegui:

  - ```npm run start```
  
  - ```npm run android```
